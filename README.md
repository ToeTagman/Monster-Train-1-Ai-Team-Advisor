WHAT IS MT ADVISOR?

MT Advisor is a fully offline-capable mobile web app built for Monster Train players. It guides you through every decision of your run — from clan and champion selection through covenants, boss strategy, store optimization, and live AI advisor calls.

Whether you’re a new player figuring out synergies or a veteran pushing Covenant 25, MT Advisor has your back with real-time AI analysis powered by Claude, one of the world’s most capable AI assistants.


FEATURES

Team Setup Wizard
Step-by-step clan, champion, path, ring, and covenant selection.

AI Advisor Tab
Ask Claude anything. It knows your full run state: deck, spells, artifacts, boss, ring, covenant — everything.

Store Optimizer
Track store pool, acquired items, and get buy/skip recommendations.

Train Tab
Manage unit and spell upgrades with visual card art.

Battle Tab
Track your boss, variant, and wave. Get fight-specific strategy.

Dashboard
Live synergy analysis, covenant penalties, and run health at a glance.

459 Embedded Card Images
Every card, unit, artifact, champion, and boss has in-game artwork baked right in — no internet needed to display them.

Offline-First
The entire app is one HTML file. All art and data is embedded. Only the AI Advisor needs a connection.

SETTING UP YOUR CLAUDE API KEY

The AI Advisor requires a Claude API key from Anthropic. This is a one-time setup that takes about 2 minutes.

STEP 1 — Create an Anthropic Account

Open your browser and go to:
console.anthropic.com

Click Sign Up and create a free account with your email.
Verify your email address.

STEP 2 — Generate Your API Key

Once logged in, tap API Keys in the left sidebar.

Tap Create Key — give it any name (e.g. "MT Advisor").

Copy the key. It starts with sk-ant- followed by a long string of characters.

Save this key somewhere safe. Anthropic only shows it once. If you lose it, just generate a new one.

STEP 3 — Add Credits

Tap Billing in the left sidebar.

Add a payment method and purchase credits. $5 is more than enough to start — each Advisor query costs a fraction of a cent.

Anthropic charges only for what you use. There are no subscriptions or monthly minimums for API access.

STEP 4 — Enter Your Key in MT Advisor

Open MT Advisor on your phone.

Tap the key icon in the top right corner.

Paste your API key and tap Save.

Your key is stored locally on your device only. It never leaves your phone except to communicate directly with Anthropic’s servers.


INSTALLING MT ADVISOR ON YOUR IPHONE

MT Advisor is a single HTML file — no App Store, no installation, no account required. You save it directly to your home screen and it behaves exactly like a native app: full screen, no browser chrome, tap-to-open.


METHOD 1 — VIA AIRDROP OR FILES APP

This is the fastest method if someone is sharing the file directly with you.

1. Receive the MT_Advisor_v9.html file via AirDrop, email, iMessage, or iCloud Drive.
2. Tap the file. It will open automatically in Safari as a full web page.
3. Tap the Share button — the box with an upward arrow at the bottom of Safari.
4. Scroll down in the share sheet and tap Add to Home Screen.
5. Give it a name (e.g. "MT Advisor") and tap Add.

The app icon now lives on your home screen. Tap it any time to launch MT Advisor in full-screen mode.


METHOD 2 — VIA SAFARI DOWNLOAD

Use this method if you received a download link.

1. Open the download link in Safari on your iPhone (not Chrome — Safari is required for Add to Home Screen).
2. If the file opens as a page, tap the Share button and proceed to step 4 above.
3. If it downloads to Files, open the Files app, tap the HTML file, and it will open in Safari.
4. Then follow the Add to Home Screen steps above.


IMPORTANT IPHONE NOTES

Use Safari only. Chrome and Firefox on iPhone cannot add pages to the home screen the same way Safari can.

Full-screen mode
Once added to your home screen, MT Advisor opens without any browser address bar or navigation — it looks and feels like a real app.

Offline use
Once installed, all card art, data, and UI works with no internet. Only the AI Advisor tab needs a connection.

Updating
When a new version is available, simply open the new HTML file in Safari and add it to your home screen again. Your run data saves automatically between sessions.

Storage
Your run state is saved in Safari’s local storage. It persists as long as you don’t clear Safari’s website data in iPhone Settings.


TIPS FOR GETTING THE MOST FROM THE ADVISOR

Keep your run state updated as you play. The Advisor reads your current deck, artifacts, boss, and ring when answering.

Ask specific questions. For example:
"Should I buy Alloy of the Ancients or save gold for a unit?"

Use the Advisor before major decisions such as store visits, boss fights, and caverns events.

Share your win condition. Tell the Advisor what you’re building toward.

If a response seems cut off, just ask again — conversation history is preserved throughout the session.
